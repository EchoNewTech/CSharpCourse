# CSharpCourse

Projekt edukacyjny dokumentujący moją naukę języka C# i platformy .NET.

Repozytorium zawiera ćwiczenia, małe aplikacje konsolowe oraz stopniowo rozwijane projekty, tworzone w celu utrwalenia podstaw i przejścia do bardziej zaawansowanych zagadnień.

## Cel

- opanowanie składni C#
- zrozumienie programowania obiektowego
- praca z .NET CLI
- nauka dobrej struktury projektu
- praktyka z systemem kontroli wersji Git

## Struktura repozytorium

Każdy katalog to osobny projekt konsolowy tworzony w trakcie nauki.  
- **/Library** – przykłady i ćwiczenia dotyczące klas, metod i struktur danych  
- **/PhoneBook** – prosty projekt konsolowy do zarządzania książką telefoniczną  
- **/bmi_calc** – projekt konsolowy do obliczania BMI  
- **/if_else** – ćwiczenia z instrukcji warunkowych (`if`, `else`)  
- **/switch** – ćwiczenia z instrukcji `switch`

## Zakres

- typy danych i operatory
- instrukcje warunkowe i pętle
- metody
- klasy i obiekty
- obsługa wyjątków
- podział na projekty i solution

## Technologie

- C#
- .NET SDK
- Git
- GitHub

## Jak uruchomić

Sklonuj repozytorium: `git clone https://github.com/EchoNewTech/CSharpCourse.git`

Wejdź do wybranego projektu: `cd nazwa_projektu`

Uruchom aplikację: `dotnet run`
